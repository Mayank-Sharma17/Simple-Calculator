# Simple Calculator 
A simple JavaScript calculator app built with HTML5, CSS3 and JavaScript.
> More **features** will be added soon.
## Guide for Keyboard keys  
| Commands         | Key Press   |
| :--------------- | :---------: |
| All clear / `C`  | `Escape`    |
| Clear / `<`      | `Backspace` |
| Brackets / `()`  | `B`         |
| Multiply / `x`   | `*`         |
| Percentage / `%` | `P`         |
| Equals-to / `=`  | `Enter`     |
